# 第七章：SOCKS5 代理协议

## 原理

SOCKS5 是一个通用的代理协议，工作在会话层 (L5)，可以代理任意 TCP/UDP 流量。

### 为什么需要 SOCKS5

```
直接连接:
  Client ──────────────── Server (可能被防火墙阻断)

SOCKS5 代理:
  Client ─── SOCKS5 Proxy ─── Server
  (本地)     (中间人)         (目标)
```

SOCKS5 相比 HTTP 代理的优势：
- 协议无关：可以代理任何 TCP/UDP 流量
- 支持 UDP 转发
- 支持多种认证方式
- 更低层：不需要理解应用层协议

### SOCKS5 协议流程

```
Client                          Proxy                          Server
  │                               │                              │
  │─── 1. 协商认证方法 ──────────→│                              │
  │←── 2. 选定认证方法 ──────────│                              │
  │                               │                              │
  │─── 3. 认证 (如需要) ────────→│                              │
  │←── 4. 认证结果 ─────────────│                              │
  │                               │                              │
  │─── 5. 连接请求 ─────────────→│── 6. 连接目标 ──────────────→│
  │←── 7. 连接结果 ─────────────│←────────────────────────────│
  │                               │                              │
  │←──────────── 8. 双向数据转发 ──────────────────────────────→│
  │                               │                              │
```

### SOCKS5 消息格式

**1. 握手请求:**
```
+----+----------+----------+
|VER | NMETHODS | METHODS  |
+----+----------+----------+
| 1  |    1     | 1-255    |
+----+----------+----------+
VER: 0x05 (SOCKS5)
METHODS: 0x00=无认证, 0x02=用户名密码
```

**2. 连接请求:**
```
+----+-----+-------+------+----------+----------+
|VER | CMD |  RSV  | ATYP | DST.ADDR | DST.PORT |
+----+-----+-------+------+----------+----------+
| 1  |  1  | X'00' |  1   | Variable |    2     |
+----+-----+-------+------+----------+----------+
CMD: 0x01=CONNECT, 0x02=BIND, 0x03=UDP ASSOCIATE
ATYP: 0x01=IPv4, 0x03=域名, 0x04=IPv6
```

**3. 连接响应:**
```
+----+-----+-------+------+----------+----------+
|VER | REP |  RSV  | ATYP | BND.ADDR | BND.PORT |
+----+-----+-------+------+----------+----------+
REP: 0x00=成功, 0x01=一般失败, 0x03=网络不可达...
```

## 示例代码

| 文件 | 说明 |
|------|------|
| `socks5_server.py` | 完整的 SOCKS5 代理服务器 |
| `socks5_client.py` | SOCKS5 客户端库和测试 |

## 运行

```bash
# 终端1: 启动 SOCKS5 代理
python3 socks5_server.py

# 终端2: 通过代理访问
python3 socks5_client.py
# 或使用 curl 测试:
curl --socks5 127.0.0.1:1080 http://httpbin.org/ip
```
